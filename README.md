# ATM Management System

## Project Description

ATM Management System is a console-based Java application that simulates ATM operations such as Deposit, Withdraw, Balance Inquiry, and Mini Statement generation.

## Features

* User Authentication
* Deposit Money
* Withdraw Money
* Balance Inquiry
* Mini Statement
* Exit Functionality

## Technologies Used

* Core Java
* OOP Concepts
* Interface
* Collections Framework (HashMap)
* Exception Handling
* Scanner Class

## How to Run

1. Compile the program:
   javac SBIBank.java

2. Run the program:
   java com.project.SBIBank
